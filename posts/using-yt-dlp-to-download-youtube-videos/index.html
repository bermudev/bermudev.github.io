Using yt-dlp to Download YouTube Videos

2023-04-02
technology , tutorial
yt-dlp , YouTube , video download , command line tool

These past few days that I have been studying and learning a lot about Artificial Intelligence and Machine Learning, I have been able to read a lot of documentation, but I have also come across some interesting videos on YouTube.

What is the problem with this? The main problem is that YouTube is known for having certain ease in deleting videos. That is why I have decided to use one of my favorite tools to save a local copy of these videos (just in case).

This tool is yt-dlp and I’m going to share some notes on how I personally use it.

What is yt-dlp?

yt-dlp is a command-line program that allows you to download videos and audio from various websites such as YouTube, Vimeo, Dailymotion, and many more. It is a fork of the popular youtube-dl project, which was taken down from GitHub in October 2020 due to a DMCA takedown notice.

It has many additional features and options compared to the original youtube-dl, such as support for more video sites, improved subtitle downloading, and post-processing options.

It is open-source and available for Windows, macOS, and Linux. It can be used from the command line, or with a graphical user interface such as youtube-dl-gui.

Using yt-dlp

Basic video download

To download a video from YouTube, you can use the following command:

yt-dlp https://www.youtube.com/watch?v=dQw4w9WgXcQ

This will download the video with the specified URL and save it to your current directory.

To download all the videos in a playlist, you can pass the url just the same as a single video and it will download ald the videos inside the playlist.

Downloading audio only

If you only want to download the audio from a video, you can use the --extract-audio and --audio-format options:

yt-dlp --extract-audio --audio-format mp3 https://www.youtube.com/watch?v=dQw4w9WgXcQ

This will download the audio from the video and save it as an MP3 file.

Downloading subtitles

To download subtitles for a video, you can use the --write-sub option:

yt-dlp --write-sub https://www.youtube.com/watch?v=dQw4w9WgXcQ

This will download the subtitles for the video in the default subtitle format and save them to a separate file.

If you want to embed all the subs into the final file you can use --embed-subs --all-subs instead.

Select specific video and audio formats

You can use the --list-formats option to view all the available formats for a video, including audio formats. To list the available formats for a video, you can use the following command:

yt-dlp --list-formats https://www.youtube.com/watch?v=dQw4w9WgXcQ

This will display a list of all available formats for the video, including the audio formats.

Once you have the list of available formats, you can choose the audio format you want to download by specifying the format code using the --format option. For example, to download the audio in the best available format, you can use the following command:

yt-dlp --format bestaudio https://www.youtube.com/watch?v=dQw4w9WgXcQ

This will download the audio in the best available format for the video. If you want to download a specific audio format, you can use the format code from the list displayed by the --list-formats option. For example, if you want to download the audio in the Opus format, you can use the following command:

yt-dlp --format 251 https://www.youtube.com/watch?v=dQw4w9WgXcQ

This will download the audio in the Opus format.

Download a list of videos in 1080p with included chapters and incrusted subs

If we take all the above into consideration, we use:

 yt-dlp -f "bestvideo[height<=1080]+bestaudio/best[height<=1080]" --embed-chapters --embed-subs --all-subs youtube_list_url

More

These are just a few examples of how I use yt-dlp. For more information on how to use yt-dlp, you can check out the official wiki.
