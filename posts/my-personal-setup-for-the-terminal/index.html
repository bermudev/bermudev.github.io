My Personal Setup for the Terminal

2023-03-07
setup , fira code , starship , nerd font
terminal , font , theme , promt , starship , nerd font

Hello everyone! It’s been a busy few weeks, I’m studying a lot about Machine Learning and I don’t have too much time to write or share my progress. However, I had to set up a new development environment and I thought, why not share a bit of my setup?

So here’s how to set up my favorite prompt and font!

The Starship Prompt

The first tool I want to introduce is the Starship prompt. This is a customizable and fast shell prompt written in Rust that displays useful information in the terminal, like useful system information, Git repository or virtual environment.

It can be installed in many ways depending on the operating system and shell. In this case, I will explain the installation of a linux system with a bash or zsh shell, for any other case you can consult the documentation.

  1. Open a terminal and run the following command:
sh -c "$(curl -fsSL https://starship.rs/install.sh)"
  1. Add the following line to your shell’s configuration file (e.g., ~/.bashrc, ~/.zshrc). Don’t forget to replace <shell> with the name of your shell (e.g., bash, zsh):
eval "$(starship init <shell>)"

Easy, right? Now you can customize the Starship prompt by editing the ~/.config/starship.toml file. This file contains a number of settings that you can tweak to your liking, you can learn more about it here.

Installing Fira Code font

You have installed Starship correctly and you have set the prompt you liked but now you see that the icons are not displayed correctly? That is probably because the font you are using does not support those icons. To fix that I always install my favorite Nerd Font: Fira Code.

Fira Code is a monospaced font that includes programming ligatures, which can make your code more readable. And what is a Nerd Font? Nerd Fonts patches developer targeted fonts with a high number of glyphs (icons). Just what we want!

So, to install this font on Linux or Windows, follow these steps:

  1. Download the Patched Fira Code font from the official repository (or any other that you like more!).
  2. Extract the font files to a folder on your system. Separate those for Windows.
  3. On Linux, copy the font files to the ~/.fonts folder. On Windows, right-click on each font file and select “Install”.

Now you can have a pretty and nice prompt with a lot of information!
